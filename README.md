# 📰 News Article Generator with Groq API

This project is a **Streamlit web application** that fetches the latest news articles, summarizes them using **Groq’s LLaMA models**, and generates a final drafted article.  
It combines **DuckDuckGo Search**, **Newspaper4k**, and **Groq LLMs** to create professional-quality news content.

---

## 🚀 Features
- 🔍 Search for recent news on any topic (via DuckDuckGo).
- 📰 Extract full article content (via Newspaper4k).
- ✍️ Summarize each article with Groq API (LLaMA 8B / 70B).
- 📑 Generate a final, well-structured article draft.
- 🎨 Interactive **Streamlit UI** with live updates.
